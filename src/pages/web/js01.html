1. 元素的js会等到DOM元素加载完毕，并且图片也加载完毕后才会执行；jQuery 会等到DOM元素加载完毕，但不会等到图片加载完毕就会执行。
  2. 原生的js如果编写了多个入口函数，后面编写的会覆盖前面的；jquery 编写多个入口函数，后面的不会覆盖前面的。
  3. jQuery.noConflict() 释放$,必须写在其它jQuery 代码之前,之后不能再用$
  4. jQuery.noConflict() 自定义访问符号: var nn = jQuery.noConflict()
  5. $(); 表示调用jQuery的核心函数
    • 接收一个函数
    • 接收一个字符串
    • 接收一个字符串选择器 $(".box1") $("#box1")
    • 接收一个代码片段 $("<p>段落</p>")
    • 接收一个DOM元素 var $span = $(span)
  6. jQuary 就是一个伪数组
  7. jQuary 的 each()可以遍历数组和伪数组
  8. jQuary 的 each() map()可以遍历数组和伪数组
    • each 静态方法 返回遍历的对象，map 返回空数组
    • each 不可以对遍历的数组处理，map 可以对遍历的数组处理，生成一个新的数组
  9. $.holdReady(true) 暂停ready事件的执行；$.holdReady(false)恢复ready事件
  10. $(".scoll").scrollTop() 获取滚动的距离
  11. $(".scoll").scrollTop(300) 设置滚动的距离
  12. $("body").scrollTop()+$("html").scrollTop() 考虑兼容性 获取网页滚动的距离
  13. $("html,body").scrollTop(300) 考虑兼容性要设置body 设置网页滚动的距离
  14. 绑定事件
      eventName(fn)
    • $("button").click(function(){})
      • on(eventName,fn)
    • $("button").on("click",function(){})
    移除事件
    • $("button").off() 移除所有事件
    • $("button").off("click") 移除click事件
    • $("button").off("click"，test1) 移除click事件的 test1事件
  15. 事件冒泡和默认行为
    • 事件从下级到上级向上传递
    • 阻止事件冒泡
        将以下事件放在子元素方法内部
      • return false
      • event.stopPropagation()
    • 阻止默认行为 比如form表单设置 submit 提交
      • return false
      • event.preventDefault()
    • 自动触发事件行为
        会有事件冒泡行为，还会触发默认行为 注意：a的click例外，不能触发默认a的跳转行为，可以通过设置a的子元素的自动点击事件，来触发a的默认跳转行为
      • $(".father").trigger("click")
        • 不会触发事件冒泡行为，不会触发按钮的默认行为
      • $(".father").triggerHandler("click")
      • 如果要触发自定义事件，自定义事件要通过on来绑定，必须通过trigger来触发，如 $("son").on("myClick",function(){}) ; $("son").trigger("myClick")
  16. 事件命名空间 通过 on 绑定，通过trigger 触发
    • $(".son").on("click.zhangSan",function(){}) 通过 trigger 来触发事件
    • 同时会触发父元素相同命名空间的函数
  17. 事件委托
      找到一个已加载元素，添加代理事件
    • $("ul").delegate("li","click",function(){})
  18. 移入移出事件
    • mouseover/mouseout 事件，子元素被移入移出会触发父元素事件； 可以用mouseenter/mouseleaver 在移入子元素不触发父元素事件
    • $(".father").hover(fn1,fn2) 同时监听鼠标移入移出事件
    • $(".father").hover(fn) 同时监听鼠标移入移出事件
  19. $(this).siblings() 找到当前元素的其它兄弟元素
  20. 监听网页的滚动 $(window).scroll(function(){})， 获取网页滚动的偏移位 $("html,body").scrollTop()
  21. $(this).children(".sub") 找到class="sub"的子元素
  22. 动画
    • $(this).show() .hide() 显示隐藏 toggle()切换
    • 滑动动画 $("div").slideDown(1000,fn) 展开, slideUp(1000,fn) 收起
    • 在需要执行动画时，先stop(),再执行；$(this).children(".sub").stop() 找到class="sub"的子元素,停止动画
    • fadeIn(time,fn) 淡入，fadeOut(time,fn) 淡出，fadeToggle(time,fn) 淡入淡出切换，fadeTo(time,0.6,fn) 淡入到0.6
    • 队列动画$(".ad").stop().slideDown(100).fadeOut(100).fadeIn(100)
    • .animate({width:500,height 500},time,"linear"，fn) 自定义动画
    • 累加属性 .animate({width:"+100",height:"+100"},time,"linear"，fn) 自定义动画
    • 关键字 .animate({width:"toggle"},time,"linear"，fn) 自定义动画
    • .delay(1000) 延迟动画执行
    • stop()停止动画
      • stop()立即停止当前动画，继续执行后续的
      • stop(false)立即停止当前动画，继续执行后续的
      • stop(false,false)立即停止当前动画，继续执行后续的
      • stop(true)立即停止当前和后续动画
      • stop(true,false)立即停止当前和后续动画
      • stop(false,true)立即完成当前动画，继续执行后续动画
      • stop(true,true)立即完成当前动画，并停止后续动画
